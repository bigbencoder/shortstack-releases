Install

  1. Unzip the download (Safari usually does this for you).
  2. Drag ShortStack.app into your Applications folder.
  3. Open it — the stack HUD docks to the bottom corner; the ▤ menu-bar icon has the controls.

Updates itself. Once installed, ShortStack checks this page's feed and offers new versions in-app — no need to come back here.
